# The Halo AI Family — Portrait Generation Guide (v3)

Generate these portraits using ComfyUI on the Strix Halo stack.
Model: SDXL Base 1.0. Style: unified compound family — one location, one look, one family.

## The Unified Look

Every portrait takes place at the **same rural compound** — a sprawling self-built
homestead with a main house, detached garage full of server racks and tools, chain-link
perimeter fence, gravel driveway, work trucks, and outbuildings. Think rural survivalist
meets IT department. Every shot uses the same film stock, same muted color grade, same
natural golden hour light hitting from the left. They all look like they belong together.

**Global style prompt suffix (append to ALL prompts):**
`shot on kodak portra 400 film, 85mm lens, shallow depth of field, golden hour natural light from the left, muted desaturated color grade, warm shadows, dusty atmosphere, rural american compound setting, photojournalistic style, candid unposed moment, ultra realistic, 8k detail`

**Negative prompt (use for ALL):**
`text, watermark, signature, blurry, low quality, deformed, cartoon, anime, cyberpunk, neon, glowing eyes, holographic, fantasy, glamorous, clean, polished, wealthy, superhero, armor, sci-fi, CGI, 3d render, studio lighting, posed, smiling, bright colors, saturated, stock photo`

**Settings:** 1024x1024, 30 steps, CFG 7.5, Euler a

## How to Generate

1. SSH into Strix Halo
2. Load SDXL Base 1.0 checkpoint
3. Use the prompts above (style suffix is already included in each prompt)
4. Apply the global negative prompt to all generations
5. Settings: 1024x1024, 30 steps, CFG 7.5, Euler a
6. Generate 4 variants of each, pick the best
7. Save to `/srv/ai/comfyui/output/family/`
